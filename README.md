# Machine-Learning-Practice

A structured collection of machine learning practice notebooks covering regression algorithms, loss function behavior, model evaluation, and algorithm implementations from scratch.  
This repository is designed to strengthen core ML concepts through hands-on experimentation and clean, intuitive visualizations.

---

##  Repository Structure

---

## Contents

### **1. Regression Algorithms**
Practical implementations of regression models, including mathematical intuition, training loops, evaluation metrics, and visualization.

| Notebook | Description |
|---------|-------------|
| **Linear_Regression_.ipynb** | Implements simple linear regression using normal equation and gradient descent. Includes error plots, fitted lines, and R² evaluation. |
| **multiple_linear_regression.ipynb** | Regression with multiple predictors, feature scaling, coefficient interpretation, and scoring. |
| **Polynomial_linear_LR.ipynb** | Builds polynomial regression using feature transformation to model nonlinear relationships. |
| **Loss_function_calculation.ipynb** | Step-by-step demonstration of Mean Squared Error (MSE), gradients, and optimization behavior. |
| **My_own_customized_LR_class.ipynb** | A complete implementation of Linear Regression from scratch using Python OOP (fit, predict, score functions). |
